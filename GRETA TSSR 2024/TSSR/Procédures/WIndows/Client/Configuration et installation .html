Configuration et installation du Client WIN_10_PRO

Configuration du nom de machine et fixage d’IP sur une machine client windows 10

 

Après avoir installé Windows 10, nous allons renommer la machine bloquer les mises à jour de Windows update pendant 7 jours, et fixer les IP en statique, étant donné que la machine redémarre lors du changement de nom, je vais commencer par fixer les IP :

 

Ouvrir la zone de recherche Windows en tapant « ncpa.cpl » puis appuyer sur entrer :

1 -Dans connexions réseau sélectionner la carte à paramétrer

2- cliquer sur propriétés

3- Sélectionner le protocole IPv4

4- cliquer sur propriétés

 

1- ouvrir en parallèle une fenêtre d’exécution (touche WIN+R) taper cmd puis entrer.

2- taper « ipconfig » dans la fenêtre de commande (noire) pour y repérer l’IPv4 (encadré rouge)

3- dans les propriétés de : protocole internet sélectionner l’option « Utiliser l’adresse IP suivante »

4- entrer manuellement les DNS également (1.1.1.1//1.0.0.1)

Cocher valider en quittant.

 

Nous allons maintenant bloquer les mises à jour windows update :

 

Dans la zone de recherche taper « Windows Uptate » puis ouvrir ensuite le menu :

Ensuite il faut simplement cliquer suspendre les mises à jour pendant 7 jours 

(Ici je dois appliquer mes mises à jour pour pouvoir les re-bloquer)

 

On va configurer le pare-feu pour autoriser les demandes d’échos venant des machines distantes :

-1 taper pare-feu dans la zone de recherche Windows

2- sélectionner pare feu […] fonctions avancées de sécurité

3- choisir les règles de trafic entrant 

4- rechercher diagnostics de réseau […] demande d’échos ICMP (IPv4)

Sélectionner les deux règles et les activer.

Refaire la manipulation pour les règles de trafic sortant.
