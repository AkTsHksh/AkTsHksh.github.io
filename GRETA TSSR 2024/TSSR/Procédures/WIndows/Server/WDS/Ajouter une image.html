AJOUTER UNE IMAGE DE CAPTURE WINDOWS

 

 

Après avoir créé ce fichier de démarrage on va devoir en créer un second qui sera une image de capture en faisant un clic droit sur le fichier créé juste avant.

Voici le détail ordonné des étapes pour créer celui-ci.

Si l’on possède déjà un fichier boot il est plus sage de le renommer (par exemple capture.wim

 

Il faudra absolument cocher la case « Ajouter une image au serveur de déploiement Windows » à la fin de la création de ce fichier avant de fermer la boite de dialogue

NOTE : EN CLIQUANT AVEC LE CLIC DROIT SUR LES BOOTS.WIM, ON PEUT CHOISIR DE LES ACTIVER OU LES OU BIEN LES METTRE HORS LIGNE (IL FAUDRA PENSER A LES REACTIVER SELON LES BESOINS).

 

Il faudra également, dans le dossier image d’installation, ajouter un groupe d’images, que l’on appellera « Capture Windows »
