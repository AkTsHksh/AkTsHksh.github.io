PARAMETRAGE DHCP

 

Cliquer sur le serveur « DHCP » ou « outils » puis « gestionnaire DHCP »

  1. sélectionner le serveur cible
  2.  cliquer sur gestionnaire DHCP

Une fois dans le gestionnaire, dérouler le serveur, puis clic droit sur nouvelle étendue pour la créer et ouvrir les paramètres de configuration.

 

Il faudra premièrement lui donner un nom et une description
