Package GLPI

On va télécharger le fichier d'install de Glpi dans le dossier tmp pour qu'il soit effacer quand on éteindra la machine

Tout d'abord on se met dans le répertoire tmp pour télécharger l'archive 

cd /tmp/

https://glpi-project.org/fr/telecharger-glpi/

Pour télécharger sur la Machine on copie l'adresse du lien et on fais wget

 

wget https://github.com/glpi-project/glpi/releases/download/10.0.7/glpi-10.0.7.tgz

 

 

On vérifie le Fichier Glpi-10.0.7 à été téléchargé

 

 

Le dossier Glpi-10.0.0.7 est compressé (tgz) donc on va le décompresser

 

tar -xvzf glpi-10.0.7.tgz

Sans le v c'est non verbeux et on vérifie si il est décompressé 

 

Maintenant on va déplacer le fichier Glpi dans le dossier html

 

mv glpi /var/www/html/

 

Puis on vérifie si il a bien été déplacé

 

ls /var/www/html/

 

 

Enfin on va changer les permissions pour que le serveur web apache ait accès à ces fichiers

on se déplace dans le dossier html

 

cd /var/www/html/

 

et on regarde les détail et les droit des dossier

 

ls -la

 

 

on change les droit du dossier et on entre l'user  ww-data dansle groupe ww-data

 

chown -R www-data:www-data /var/www/html/

 

Puis on change les Droit

 

chmod -R 750 /var/www/html/glpi
