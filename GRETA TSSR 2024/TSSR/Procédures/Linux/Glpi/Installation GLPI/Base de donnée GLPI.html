Base de donnée GLPI

Tout d'abord il faut rentrer dans le terminal de mariadb afin de créer notre base de donnée GLPI

 

mysql -u root -p

 

Puis on créer la base de donnée

ici on choisie le nom de la base “glpi”

Attention sur les commande suivante il ne faut pas oublier les “;” 

create database glpi;

 

 

Ensuite on créer notre utilisateur (modifier "user" ainsi que “motdepasse”)

 

create user glpiuser@localhost identified by "Greta37!";

 

 

Après cela on donne tous les privilèges à cette utilisateur de la base de donnée 

grant all privileges on glpi.* to glpiuser@localhost;

 

 

Enfin on affecte les changements de privilèges aux utilisateurs 

flush privileges;

 

 

Pour quitter le terminal 

QUIT

 

 

La base de données est installé
