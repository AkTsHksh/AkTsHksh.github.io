Interface graphique GLPI

Tout d'abord connectez vous sur l'interface glpi

http://ipdevotreserveur/glpi

Ici l'adresse est 172.16.1.100

 

http://172.16.1.100/glpi

 

Indiquez votre identifiant et mot de passe (par default : id=glpi et mdp=glpi)

 

 

Pour la suppression du fichier install.php rendez-vous dans votre dossier install

 

cd /var/www/html/glpi/install

 

Ensuite supprimez le fichier install.php

 

rm install.php

 

 

Rechargez la page de votre navigateur la ligne disant de supprimer le fichier install.php devrait avoir disparue

 

 

Ensuite dans l'onglet "Administration" puis "Utilisateurs" sur le côté gauche vous pouvez changer les mot de passe par défault ainsi que les utilisateurs du GLPI
