commandes utiles LINUX

MODIFICATION D’UTILISATEURS OU DE GROUPES :

 

Créer un groupe :

groupadd « nom_groupe »

 

Créer un utilisateur (ce qui crée un groupe du même nom)

adduser « nom_utilisateur »

 

Ajouter un utilisateur et le placer dans un groupe existant

adduser « nom_utilisateur » - gid id_group

 

Ajouter un utilisateur à un groupe

adduser « nom_utilisateur » « nom_groupe » (oubien usermod -a -G « nom_groupe » « nom_utilisateur »)

 

Supprimer un utilisateur

deluser « nom utilisateur »

 

Supprimer l’utilisateur ainsi que son dossier personnel

deluser –remove-home « nom_utilisateur »

 

Modifier le mot de passe d’un utilisateur 

passwd « nom_utilisateur »


 

 

Vérification de l’intégrité de fichiers :

Depuis un terminal avec la commande :

sha1sum « nom_du_fichier »

 

Vérification du code qui est un fichier en « .sum » (issu du site/fourni avec le fichier ISO)

cat « nom_du_fichier »

Si les deux sont identiques, alors l’ISO est intègre.

 

Observation des répertoires et des fichiers :

 

Afficher la taille occupée par le répertoire personnel d’un utilisateur :

du -hs /home/« username »

 

Afficher la taille des répertoires contenus dans le répertoire personnel d’un utilisateur :

Du -h -d 1 /home/« username »

 

Afficher les infos de base sur un fichier :

ls -hs « filename »

 

Afficher un max d’infos sur un fichier (contenu Inode)

stat « filename »

 

Pour un fichier donné, afficher le nom du système de fichier et le point de montage :

df -T « filename »cat « filename »

 

Afficher en hexa & ASCII les données contenues dans un fichier :

hexdump -C « finlename »

 

Manipulation de fichiers :

Créer un fichier :

touch « filename »

 

Editer le fichier créé et y ajouter du contenu avant de l’enregistrer :

nano « filename »

 

Créer une copie du fichier :

cp filename filename_copy

 

Renommer un fichier

mv filename newfilename

 

Créer un répertoire et déplacer le fichier dans ce répertoire :

mkdir « repertoirename »
cp « filename »/« repertoirename »

 

Supprimer la copier du fichier

rm repertoirename

 

Permissions

Afficher : nom ru propriétaire, groupe taille et permissions du fichier :

Ls -hs « filename »

 

Modifier le nom du groupe

Chown :« grpname » « filename »
