Router DHCP

ACTIVER LE DHCP SUR UN ROUTER CISCO

 

Acitver le mode admin et changer le nom

Router>enable

Router#configure terminal 

Enter configuration commands, one per line. End with CNTL/Z.

Router(config)#hostname ROUTER_DHCP

 

Alumer l’interface

ROUTER_DHCP(config)#interface gigabitEthernet 0/1

ROUTER_DHCP(config-if)#no shutdown

%LINK-5-CHANGED: Interface GigabitEthernet0/1, changed state to up

%LINEPROTO-5-UPDOWN: Line protocol on Interface GigabitEthernet0/1, changed state to up

 

Fixer l’ip de l’interface désirée

ROUTER_DHCP(config-if)#ip address 172.16.255.254 255.255.0.0

ROUTER_DHCP(config-if)#exit

 

Nommer le dhcp et donner les adresses distribuables (+ no ip domain-lookup)

ROUTER_DHCP(config)#ip dhcp pool TEST

ROUTER_DHCP(dhcp-config)#net

ROUTER_DHCP(dhcp-config)#network 172.16.0.0 255.255.0.0

ROUTER_DHCP(dhcp-config)#dns-server 1.1.1.1

ROUTER_DHCP(dhcp-config)#default-router 172.16.255.254

ROUTER_DHCP(dhcp-config)#^Z

ROUTER_DHCP#conf t

ROUTER_DHCP(config)#no ip domain lookup

ROUTER_DHCP(config)#exit

ROUTER_DHCP#

%SYS-5-CONFIG_I: Configured from console by console


 

 

Vérification & afficher les ip distribuées par ce router dans son interface

ROUTER_DHCP#show ip dhcp binding
